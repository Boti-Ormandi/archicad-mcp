# Surface Assignment PR - Implementation Plan

## Overview

**Goal:** Add surface material assignment to `GetDetailsOfElements` and `SetDetailsOfElements` for Walls, Slabs, Column Segments, and Beam Segments.

**Scope decisions:**
- **Included element types:** Wall, Slab, Mesh, ColumnSegment, BeamSegment (all have existing GetDetailsOfElements coverage)
- **Excluded element types:** Roof, Shell (no GetDetailsOfElements support yet; Roof has complex per-segment/per-edge surfaces)
- **Clear override support:** Pass `-1` to revert surface to building material default
- **Grasshopper plugin:** Out of scope - plugin can consume new fields via raw JSON; typed C# classes can be added in a separate PR by Grasshopper maintainers

**Files to modify:**
1. `archicad-addon/Sources/ElementCommands.cpp`
2. `archicad-addon/Sources/RFIX/Images/CommonSchemaDefinitions.json`

**Auto-updated (no manual changes needed):**
- `docs/archicad-addon/*` - Auto-generated by `GenerateDocumentation` command after rebuild
- `Test/ExpectedOutputs/*.output` - Snapshot files update when tests run

**Surface fields by element type:**

| Element Type | Fields | Access Pattern |
|--------------|--------|----------------|
| Wall | `refMat`, `oppMat`, `sidMat` | `elem.wall.refMat` |
| Slab | `topMat`, `botMat`, `sideMat` | `elem.slab.topMat` |
| Mesh | `topMat`, `botMat`, `sideMat` | `elem.mesh.topMat` (same pattern as Slab) |
| ColumnSegment | `extrusionSurfaceMaterial`, `endsMaterial` | `elem.columnSegment.extrusionSurfaceMaterial` |
| BeamSegment | `leftMaterial`, `topMaterial`, `rightMaterial`, `bottomMaterial`, `endsMaterial` | `elem.beamSegment.leftMaterial` |

**User workflow difference:**
- Wall/Slab/Mesh: Get GUID via `GetElementsByType("Wall")`, `GetElementsByType("Slab")`, or `GetElementsByType("Mesh")`
- ColumnSegment/BeamSegment: Get GUID via `GetSubelementsOfHierarchicalElements` on a Column/Beam

## Reference: Archicad API DevKit

The cloned DevKit at `archicad-api-devkit/docs/` contains the authoritative struct definitions:

| Struct | DevKit Reference | Key Surface Fields |
|--------|------------------|-------------------|
| `API_WallType` | `struct_a_p_i___wall_type.html` | `refMat`, `oppMat`, `sidMat` (all `API_OverriddenAttribute`) |
| `API_SlabType` | `struct_a_p_i___slab_type.html` | `topMat`, `botMat`, `sideMat` (all `API_OverriddenAttribute`) |
| `API_MeshType` | `struct_a_p_i___mesh_type.html` | `topMat`, `botMat`, `sideMat` (same pattern as Slab) |
| `API_ColumnSegmentType` | `struct_a_p_i___column_segment_type.html` | `extrusionSurfaceMaterial`, `endsMaterial` |
| `API_BeamSegmentType` | `struct_a_p_i___beam_segment_type.html` | `leftMaterial`, `topMaterial`, `rightMaterial`, `bottomMaterial`, `endsMaterial` |
| `API_Element` (union) | `union_a_p_i___element.html` | Contains `wall`, `slab`, `mesh`, `columnSegment`, `beamSegment` members |

**API_OverriddenAttribute structure:**
- `IsOverridden` (bool) - whether the surface is overridden from building material default
- `Value` (API_AttributeIndex) - the surface attribute index when overridden

**Override behavior:**
- When `IsOverridden = true`: Element uses the explicit surface from `Value`
- When `IsOverridden = false`: Element inherits surface from its building material
- In JSON API: Pass surface index >= 0 to set, pass `-1` to clear (revert to building material default)

**Useful DevKit searches:**
```bash

# Check specific struct
cat archicad-api-devkit/docs/struct_a_p_i___wall_type.html | grep -A2 "refMat\|oppMat\|sidMat"
```

#### 2.2.1 Wall (modify existing case, around line 822)

**Find the existing Wall case and add surface handling after geometry:**

```cpp
case API_WallID: {
    const GS::ObjectState* begCoordinate = typeSpecificDetails->Get ("begCoordinate");
    if (begCoordinate != nullptr) {
        elem.wall.begC = Get2DCoordinateFromObjectState (*begCoordinate);
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, begC);
    }
    const GS::ObjectState* endCoordinate = typeSpecificDetails->Get ("endCoordinate");
    if (endCoordinate != nullptr) {
        elem.wall.endC = Get2DCoordinateFromObjectState (*endCoordinate);
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, endC);
    }
    if (typeSpecificDetails->Get ("height", elem.wall.height)) {
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, height);
    }
    if (typeSpecificDetails->Get ("offset", elem.wall.offset)) {
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, offset);
    }

    switch (elem.wall.type) {
        case APIWtyp_Trapez: {
            if (typeSpecificDetails->Get ("begThickness", elem.wall.thickness)) {
                ACAPI_ELEMENT_MASK_SET (mask, API_WallType, thickness);
            }
            if (typeSpecificDetails->Get ("endThickness", elem.wall.thickness1)) {
                ACAPI_ELEMENT_MASK_SET (mask, API_WallType, thickness1);
            }
        } break;
        default:
        break;
    }

    // Surface materials (pass -1 to clear override and revert to building material default)
    Int32 refMatIndex;
    if (typeSpecificDetails->Get ("refMat", refMatIndex)) {
        if (refMatIndex < 0) {
            elem.wall.refMat.IsOverridden = false;
        } else {
            elem.wall.refMat.IsOverridden = true;
            elem.wall.refMat.Value = ACAPI_CreateAttributeIndex (refMatIndex);
        }
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, refMat);
    }

    Int32 oppMatIndex;
    if (typeSpecificDetails->Get ("oppMat", oppMatIndex)) {
        if (oppMatIndex < 0) {
            elem.wall.oppMat.IsOverridden = false;
        } else {
            elem.wall.oppMat.IsOverridden = true;
            elem.wall.oppMat.Value = ACAPI_CreateAttributeIndex (oppMatIndex);
        }
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, oppMat);
    }

    Int32 sidMatIndex;
    if (typeSpecificDetails->Get ("sidMat", sidMatIndex)) {
        if (sidMatIndex < 0) {
            elem.wall.sidMat.IsOverridden = false;
        } else {
            elem.wall.sidMat.IsOverridden = true;
            elem.wall.sidMat.Value = ACAPI_CreateAttributeIndex (sidMatIndex);
        }
        ACAPI_ELEMENT_MASK_SET (mask, API_WallType, sidMat);

} break;

## Summary

Adds ability to read and write surface materials on Walls, Slabs, Column Segments, and Beam Segments via `GetDetailsOfElements` and `SetDetailsOfElements`.

## Changes

**Wall fields:**
- `refMat` - reference line side surface
- `oppMat` - opposite side surface
- `sidMat` - edge surface

**Slab fields:**
- `topMat` - top surface
- `botMat` - bottom surface
- `sideMat` - edge surface

**Mesh fields:**
- `topMat` - top surface
- `botMat` - bottom surface
- `sideMat` - edge surface

**ColumnSegment fields:**
- `extrusionSurfaceMaterial` - main extrusion surface
- `endsMaterial` - end cap surfaces

**BeamSegment fields:**
- `leftMaterial` - left side surface
- `topMaterial` - top surface
- `rightMaterial` - right side surface
- `bottomMaterial` - bottom surface
- `endsMaterial` - end cap surfaces

## Implementation Notes

- Follows existing pattern for attribute handling (see `layerIndex`)
- Surfaces only returned in GET when `IsOverridden == true`
- Pass `-1` in SET to clear override and revert to building material default
- Uses `ACAPI_CreateAttributeIndex()` for index conversion
- Column/Beam segments are accessed via their own GUIDs (from `GetSubelementsOfHierarchicalElements`)
- Works with both basic structure and composite structure elements (overrides outermost surface)

## New Schemas

- `SlabSettings` - for SetDetailsOfElements
- `MeshSettings` - for SetDetailsOfElements
- `ColumnSegmentDetails` / `ColumnSegmentSettings`
- `BeamSegmentDetails` / `BeamSegmentSettings`

## Usage Example

```python
# For walls/slabs - direct element access
walls = GetElementsByType("Wall")
SetDetailsOfElements([{
    "elementId": walls[0]["elementId"],
    "details": {"typeSpecificDetails": {"refMat": 5}}
}])

# Clear override - revert to building material default
SetDetailsOfElements([{
    "elementId": walls[0]["elementId"],
    "details": {"typeSpecificDetails": {"refMat": -1}}
}])

# For columns/beams - access via segments
columns = GetElementsByType("Column")
segments = GetSubelementsOfHierarchicalElements([{"elementId": columns[0]["elementId"]}])
SetDetailsOfElements([{
    "elementId": segments["subelements"][0]["columnSegments"][0]["elementId"],
    "details": {"typeSpecificDetails": {"extrusionSurfaceMaterial": 5}}
}])
